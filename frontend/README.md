# I have created the project by using this command:
"""npm create vite@latest . -- --template react-ts"""
- This is the modern, recommended way to create a React + TypeScript project with Vite.
- . → Important: Create the project in the current folder (your frontend/ folder). we use (.) so it doesn’t create an extra nested folder.
- -- --template react-ts → The two dashes (--) separate Vite's options. It tells Vite to use the React + TypeScript template.
